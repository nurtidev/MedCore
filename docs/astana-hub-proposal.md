# Предложение для Astana Hub

## Название проекта

MedCore — медицинская SaaS-платформа для клиник Казахстана

## Краткое описание

MedCore — это модульная платформа для цифровизации операционных процессов клиник и медицинских центров. Решение объединяет управление доступом и ролями, онлайн-оплаты, интеграции с государственными и частными медицинскими системами, а также BI-аналитику по KPI в единой архитектуре.

Проект ориентирован на рынок Республики Казахстан и учитывает требования к защите персональных данных, интеграциям с локальными сервисами и особенностям медицинского бизнеса.

## Какую проблему решает проект

Сегодня клиники и медицинские сервисы часто работают в разрозненной ИТ-среде:

- доступы пользователей и роли настраиваются вручную
- платежи проходят через внешние каналы без единого контроля в системе
- интеграции с лабораториями и агрегаторами требуют ручного переноса данных
- управленческая аналитика собирается фрагментарно и с задержкой

Это создаёт риски для безопасности данных, повышает нагрузку на персонал и снижает управляемость бизнеса.

## Предлагаемое решение

MedCore предлагает единую платформу со следующими ключевыми модулями:

### 1. Аутентификация и ролевая модель

- JWT-аутентификация с access и refresh токенами
- RBAC-модель для врачей, координаторов, администраторов и суперпользователей
- безопасное хранение чувствительных данных, включая ИИН
- централизованный контроль доступа для всех модулей платформы

### 2. Онлайн-оплаты и биллинг

- генерация ссылок на оплату
- интеграция с Kaspi Pay и Stripe
- обработка webhook-событий по оплатам
- SaaS-подписки клиник
- генерация PDF-инвойсов и документов

### 3. Интеграции с государственными и частными медицинскими системами

- eGov API
- DAMUMED
- iDoctor
- лаборатории Олимп и Invivo

Платформа поддерживает адаптерный подход, webhooks, retry-механизмы, circuit breaker и журналирование интеграционных операций.

### 4. BI-аналитика и KPI

- аналитика выручки
- загрузка врачей и расписания
- no-show rate
- конверсионные воронки
- подготовка данных для управленческих решений в near real-time

## Технологическая реализация

Проект реализован как микросервисная система:

- Backend: Go
- API Gateway: Go + reverse proxy + gRPC auth validation
- Frontend: Vue 3 + TypeScript
- Основная БД: PostgreSQL
- Кэш: Redis
- События: Kafka
- Аналитика: ClickHouse
- PDF: Gotenberg
- Контейнеризация: Docker Compose

Такой стек обеспечивает производительность, масштабируемость и удобство дальнейшего развития платформы.

## Преимущества проекта

- ориентация на рынок Казахстана и локальные интеграции
- единая архитектура вместо набора несвязанных решений
- снижение ручного труда администраторов и координаторов
- повышение прозрачности финансовых и операционных показателей
- готовность к масштабированию как B2B SaaS-платформы

## Ожидаемый эффект для рынка

Внедрение MedCore позволит клиникам:

- ускорить обработку записей, оплат и результатов анализов
- сократить количество ручных ошибок и операционных потерь
- повысить безопасность медицинских данных
- улучшить контроль за выручкой, расписанием и загрузкой персонала
- быстрее принимать управленческие решения на основе KPI

- этап 1: пилотный запуск core-модулей
- этап 2: подключение внешних интеграций
- этап 3: расширение аналитики и продуктовых сценариев
- этап 4: production rollout для клиник и партнёров

## Заключение

MedCore — это практическая основа для современной медицинской цифровой платформы в Казахстане. Решение закрывает как базовые инфраструктурные задачи, так и прикладные бизнес-потребности клиник: безопасность, биллинг, интеграции и аналитика.
