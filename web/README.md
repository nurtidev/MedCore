# MedCore Web

Frontend application for the MedCore platform.

## Stack

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Vue I18n
- ECharts
- Vitest

## Commands

```bash
npm install
npm run dev
npm test
npm run build
```

## Default Local URL

```text
http://localhost:5173
```

For full project setup and backend integration details, see the root [README](../README.md).
